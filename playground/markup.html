Note how adding layers darkens shadows; you typically want to counter-act this by reducing base-layer opacity:

Apply Layer Scaling

Note how adding layers and reducing opacity doesn't really change anything -- this is where layer scaling comes in:

Default:
shadow-lg

shadow-sm
shadows-4
shadow-opacity-5
shadow-y-[3px]
shadows-scale-3

shadow-sm
shadows-7
shadow-opacity-5
shadow-y-[3px]
-shadow-spread-[10px]
shadows-scale-1.5

Apply Easing to Layer Scaling

Finally, you may want to apply easing for a more fluid/less linear scaling effect.

Default:
shadow-lg

shadow-sm
shadows-4
shadow-opacity-5
shadow-y-[3px]
shadows-scale-3
shadows-ease-in

shadow-sm
shadows-4
shadow-opacity-5
shadow-y-[3px]
shadows-scale-3
shadows-ease-out
